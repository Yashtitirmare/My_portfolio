## üì¶ Installation

To run this project locally, follow these steps:

### Prerequisites

Make sure you have **Node.js** and **npm** installed. If not, you can download them from [Node.js official website](https://nodejs.org/).

### Steps

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/your-username/your-portfolio.git
   
2. Navigate to the project directory:

   ```bash
   cd your-portfolio
   
3. Install the dependencies using npm or yarn:

   ```bash
     npm install
     Or if you're using Yarn:
     yarn install

4. Run the development server:

   ```bash
     npm run dev
     Or if you're using Yarn:
     yarn dev

## ‚úâÔ∏è Contact Form Setup

If you want to set up the email feature, you will need to configure EmailJS.

1. Go to EmailJS and create an account.
2. Set up a new email service (e.g., Gmail).
3. Create an email template for the contact form.
4. In the .env file, add your EmailJS credentials:
   ```bash
   VITE_EMAILJS_USER_ID=your_emailjs_user_id
   VITE_EMAILJS_SERVICE_ID=your_email_service_id
   VITE_EMAILJS_TEMPLATE_ID=your_email_template_id
